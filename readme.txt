== Description ==

O plugin [Beezzer Club](http://pt.beezzer.com/about/pluginWordpress/ "Beezzer Club") permite que você crie um clube 
de consumidores e clientes dentro do seu Blog, usando o clube criado no [Beezzer](http://pt.beezzer.com/ "Beezzer"). 
É possível que seus usuários criem tickets para resolverem problemas, tirarem dúvidas, darem dicas ou simplemente 
trocarem informações com outros consumidores.  
  
**Pré-requisitos:**  
- Wordpress a partir do 2.1.x  
- Módulo php-curl  
  
**Demonstração:**  
Uma demonstração está disponível no Blog do próprio Beezzer: [http://blog.beezzer.com/clube](http://blog.beezzer.com/clube "Blog do Beezzer").

== Installation ==

1. Crie um clube no http://pt.beezzer.com com a conta que será usada no plugin.
2. Instale o plugin descompactando o arquivo zip no seu wp-content/plugins.
3. Ative o plugin na página de administração "Plugins Management" do seu Blog.
4. Um novo ítem "Beezzer Club Config" estará disponível no menu. Acesse-o para inserir dados da sua conta e clube.
5. Crie uma página chamada "Clube do Blog". Pronto! O clube estará disponível na página criada.
